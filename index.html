What I Do

With over 4+ years of experience in game development, I take ideas from the drawing board all the way to published, playable games. Whether I am working on a mobile app, a web-based game, or exploring Virtual Reality, my goal is always to create a seamless experience for the player.


My Approach:

  • Building the Foundation: I write clean, modular code that acts as the sturdy backbone for any game, making it easy to add new features later.
  • Performance First: I optimise games so they look beautiful without draining battery life or lagging on older devices.
  • Connecting Players: I build the networking systems that allow for multiplayer gameplay, live leaderboards, and real-time updates.
  • Creating Tools: I build custom software tools that help my whole team work faster and more efficiently.
